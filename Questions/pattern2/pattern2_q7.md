# 1.7 MPLSアンダーレイ

Global SP #1を、以下の要件を満たすように構成しなさい。

- ルータ **R1、R2、R3、R4、R5、R6** にOSPFが設定されていることを確認しなさい。  
  また、Global SP #1内のすべての内部向けリンクで、OSPFネイバー関係が確立されていることを確認しなさい。

- Global SP #1のコアネットワークで、**ラベルスイッチング**が設定されていることを確認しなさい。

- すべてのルータで、**Loopback0をLDPルータIDとして使用**しなければならない。

- 未承認のLDPピアがセッションを確立することを防止し、さらに偽装されたTCPメッセージをブロックすることで、Global SP #1を保護しなさい。

  - R1およびR2では、複数のLDPピアを対象とするグループに対して、**LDP MD5認証をグローバルに設定**しなさい。
  - R1およびR2では、**すべてのLDPピアにパスワードを必須**としなければならない。
  - R3、R4、R5、R6では、**LDP MD5認証をピア単位で設定**しなさい。
  - Global SP #1内のすべてのLDP認証で、パスワードとして **`CCIE!nfr4`** を使用しなさい。

- Global SP #1内のどのルータのLSDBにも、**Type 2 LSAが存在してはならない**。

- Global SP #1内のLSAには、**いかなるトランジットリンクの情報も含まれてはならない**。
